## 🌟 Features

- 🔍 **Semantic Search** — finds movies by meaning, not just keywords
- 🤖 **Multi-LLM Support** — switch between Mistral, Claude, OpenAI, or HuggingFace by changing one line in `.env`
- 💬 **Conversation History** — tracks your full Q&A session
- 🏆 **Top Rated Movies** — sidebar displays highest rated movies from the database
- 🎨 **Cinematic Dark UI** — custom gold-and-black theme with smooth animations
- ⚡ **Fast Retrieval** — MongoDB Atlas vector search returns results in milliseconds
- 📽️ **Source Transparency** — shows exactly which movies were used to generate each answer
